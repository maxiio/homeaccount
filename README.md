
#  homeaccount
家庭记账系统（ssh+shiro+solr）

##  搭建系统步骤
  >  需要准备  
  
   1.  准备**tomcat8**版本  
   2.  准备**eclipse**带有sts的最好\[我使用的eclipse版本是**Neon**\]  
   3.  Maven-3.5.3  
   4.  jdk1.8  
   5.  mysql-5.1.55-community  
   6.  Solr-4.10.3
   7.    Mysql图形界面：Sqlyog
   8.  浏览器：谷歌浏览器
   
   >  下载项目  
   
   [github地址](https://github.com/jianghuxiaoao/homeaccount)  
   
   >  在Mysql中创建数据库：account  
   >  在eclipse中导入下载的项目
